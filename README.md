# Tic-Tac-Toe

A simple, interactive Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Features
- Classic Tic-Tac-Toe gameplay
- Dark and Light mode support with local storage memory
- Responsive design
- Clean, modern UI

## How to Play
1. Open `game.html` in your web browser.
2. Click on any empty cell to place your move (Player X goes first).
3. The game alternates turns between Player X and Player O.
4. The first player to get 3 in a row (horizontally, vertically, or diagonally) wins!
5. Use the "Restart Game" button to play again.
6. Toggle between Dark and Light mode using the button in the top right corner.
